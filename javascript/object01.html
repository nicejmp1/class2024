01. 변수 : if문

if문은 프로그래밍에서 조건문을 처리하는 데 사용됩니다. 일반적으로 조건을 평가하여 결과가 참(True)이면 특정한 코드 블록이 실행됩니다. 조건이 거짓(False)이면 해당 코드 블록은 건너뛰고, 프로그램의 흐름은 다음 문장으로 이동합니다. if문은 단독으로 사용될 수 있으며, 필요에 따라 else나 elif(else if의 축약형)문과 함께 사용하여 여러 조건을 처리할 수 있습니다. 이를 통해 프로그램은 다양한 조건에 따라 다른 동작을 수행할 수 있습니다. 

{
    //true, "문자열", 숫자, ,"1", "2", [배열], {객체}
    //false. ""(빈문자열), 0, null, undefined

    if (0) {
        console.log("실행되었습니다. (true)");
    } else {
        console.log("실행되었습니다. (false)");
    }
}
결과 확인하기
실행되었습니다. (false)
