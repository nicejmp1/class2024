01. 산술 연산자

산술 연산자는 수학적 계산을 수행하는 연산자로, 주로 숫자 데이터를 다룰 때 사용됩니다. 이 연산자들은 덧셈(+), 뺄셈(-), 곱셈(*), 나눗셈(/), 나머지(%) 등을 포함하며, 숫자 데이터 간의 산술적인 작업을 수행하여 결과 값을 반환합니다. 자바스크립트에서는 산술 연산자를 사용하여 숫자 데이터를 계산하고 다양한 수학적 작업을 처리할 수 있습니다. 

{
    let sum = 5 + 3; // 덧셈
    let difference = 8 - 4; // 뺄셈
    let product = 2 * 6; // 곱셈
    let quotient = 10 / 2; // 나눗셈
    let remainder = 11 % 3; // 나머지
    let power = 2 ** 3; // 거듭제곱                    
}

02. 할당 연산자

할당 연산자는 변수에 값을 할당할 때 사용되는 연산자입니다. 주로 변수에 값을 초기화하거나 변수에 새로운 값을 할당할 때 사용됩니다. 예를 들어, = 연산자는 변수에 값을 할당하는 가장 기본적인 할당 연산자이며, 다른 할당 연산자들은 이를 보완하여 복합적인 할당 작업을 수행합니다. 예를 들어, += 연산자는 현재 변수 값에 다른 값을 더한 후 그 결과를 변수에 다시 할당하는데 사용됩니다. 자바스크립트에서는 다양한 할당 연산자들을 사용하여 변수의 값을 간편하게 변경하고 조작할 수 있습니다. 

{
    let x = 10;
    x += 5; // x = x + 5;
    x -= 3; // x = x - 3;
    x *= 2; // x = x * 2;
    x /= 4; // x = x / 4;
    x %= 3; // x = x % 3;                                       
}
결과 확인하기

03. 비교 연산자

비교 연산자는 두 값 또는 표현식의 비교를 수행하여 참(True) 또는 거짓(False)을 반환하는 연산자입니다. 주로 조건문과 함께 사용되어 프로그램의 흐름을 제어하는 데에 활용됩니다. 다양한 비교 연산자들이 있으며, 주로 사용되는 것은 동등 비교(==, ===), 부등 비교(!=, !==), 크기 비교(>, >=, <, <=) 등이 있습니다. 이 연산자들은 주로 숫자와 문자열 데이터의 비교, 논리적인 조건의 평가 등에 사용됩니다. 

{
    let isEqual = (10 == 5); // 동등 비교
    let isStrictEqual = (10 === '10'); // 일치 비교
    let isNotEqual = (5 != 3); // 부등 비교
    let isNotStrictEqual = (5 !== '5'); // 부등 비교 (타입 비교)
    let isGreater = (10 > 5); // 크기 비교
    let isLessOrEqual = (3 <= 3); // 작거나 같음 비교                                                          
}
결과 확인하기
false, false, true, true, true, true

04. 논리 연산자

논리 연산자는 논리적인 조건을 결합하거나 조작하는 데 사용되는 연산자입니다. 주로 부울(boolean) 값과 함께 사용되며, 두 개의 부울 값 사이에 논리적인 연산을 수행하여 새로운 부울 값을 반환합니다. 주요한 논리 연산자로는 논리 AND(&&), 논리 OR(||), 그리고 논리 NOT(!)이 있습니다. 논리 AND는 두 조건이 모두 참일 때 참을 반환하고, 논리 OR은 둘 중 하나라도 참이면 참을 반환하며, 논리 NOT은 주어진 조건의 반대를 반환합니다. 이러한 논리 연산자들은 주로 조건문과 함께 사용되어 프로그램의 흐름을 제어하거나 조건을 평가하는 데에 활용됩니다. 

{
    let andResult = (true && false); // 논리 AND
    let orResult = (true || false); // 논리 OR
    let notResult = !true; // 논리 NOT                                                                            
}
결과 확인하기
false, true, false

05. 비트 연산자

비트 연산자는 비트 수준에서 비트들을 조작하는 연산자입니다. 주로 정수형 데이터를 이진수로 표현할 때 사용되며, 비트 단위로 AND(&), OR(|), XOR(^), NOT(~), 왼쪽 시프트(<<), 오른쪽 시프트(>>) 등의 연산을 수행합니다. 각각의 비트 연산자들은 비트 단위로 값을 조작하여 다양한 목적에 사용됩니다. 예를 들어, 비트 AND 연산자는 두 값의 비트가 모두 1이면 1을 반환하고, 비트 OR 연산자는 둘 중 하나 이상의 비트가 1이면 1을 반환합니다. 비트 연산자들은 주로 저수준 프로그래밍이나 비트 수준 연산이 필요한 경우에 사용됩니다. 

{
    let bitwiseAnd = 5 & 3; // 비트 AND
    let bitwiseOr = 5 | 3; // 비트 OR
    let bitwiseXor = 5 ^ 3; // 비트 XOR
    let bitwiseNot = ~5; // 비트 NOT
    let leftShift = 5 << 1; // 왼쪽 시프트
    let rightShift = 5 >> 1; // 오른쪽 시프트
    let unsignedRightShift = -5 >>> 1; // 부호 없는 오른쪽 시프트                                                                                               
}
결과 확인하기
1, 7, 6, -6, 10, 2, 2147483645

06. 삼향 연산자

삼항 연산자는 조건 연산자라고도 불리며, 세 개의 피연산자를 가지고 조건을 평가하여 그 결과에 따라 값을 반환하는 연산자입니다. 일반적으로 조건문을 한 줄로 간결하게 표현할 때 사용됩니다. 삼항 연산자는 다음과 같은 구조를 가집니다: 조건 ? 값1 : 값2. 조건이 참이면 값1을 반환하고, 거짓이면 값2를 반환합니다. 예를 들어, age >= 18 ? '성인' : '미성년자'와 같이 사용할 수 있습니다. 

{
    let condition = true;
    let result = condition ? '참' : '거짓';                                                                                                                 
}
결과 확인하기
참
