01. 문자열(String)

문자열(String)은 텍스트 데이터를 나타내는 자료형으로, 문자들의 집합을 나타냅니다. 문자열은 작은따옴표(''), 큰따옴표(""), 백틱(``)으로 감싸여 표현됩니다. 예를 들어, "Hello, World!"와 같이 텍스트를 나타내는 것이 문자열입니다. 문자열은 자바스크립트에서 매우 중요한 자료형 중 하나이며, 다양한 문자열 처리 작업을 수행하는 데 사용됩니다. 

{
    let greeting = "Hello, World!";
    console.log(greeting); // 출력: Hello, World!                                    
}
결과 확인하기
Hello, World!

02. 숫자(Number)

숫자(Number)는 수학적인 값이나 연산을 나타내는 자료형입니다. 정수나 소수점을 포함한 수를 표현할 수 있습니다. 숫자는 자바스크립트에서 다양한 연산을 수행하는 데 사용되며, 산술 연산이나 수치적인 계산을 위해 주로 활용됩니다. 예를 들어, 42나 3.14와 같은 것이 숫자입니다. 

{
    let number = 42;
    console.log(number); // 출력: 42                                                     
}
결과 확인하기
42

03. 불리언(Boolean)

불리언(Boolean)은 참(True) 또는 거짓(False) 값을 나타내는 자료형입니다. 불리언은 주로 조건을 평가하거나 논리적인 연산에 사용됩니다. 예를 들어, 비교 연산의 결과나 조건문의 조건식 등에 사용됩니다. 자바스크립트에서는 불리언 값은 true와 false로 표현됩니다. 이를 이용하여 프로그램의 흐름을 제어하거나 조건을 판단할 때 활용됩니다. 

{
    let isTrue = true;
    console.log(isTrue); // 출력: true                                                                           
}
결과 확인하기
true

04. null

null은 "값이 없음"을 나타내는 자료형입니다. null은 명시적으로 변수에 값이 없음을 할당할 때 사용됩니다. 예를 들어, 어떤 변수가 초기화되지 않았거나 비어있는 경우에 해당 변수에 null을 할당하여 "값이 없음"을 나타낼 수 있습니다. null은 자바스크립트에서 객체가 아닌 원시 값 중 하나입니다. 

{
    let nullValue = null;
    console.log(nullValue); // 출력: null                                                                                              
}
결과 확인하기
null

05. undefined

undefined는 변수가 선언되었지만 아직 값을 할당받지 않았거나 변수에 접근할 때 값이 없음을 나타내는 자료형입니다. undefined는 변수가 초기화되지 않았을 때 자동으로 할당되거나, 존재하지 않는 객체의 속성에 접근했을 때 반환되기도 합니다. 일반적으로 프로그램에서 undefined를 직접 할당하지는 않습니다. undefined는 자바스크립트에서 null과 함께 "값이 없음"을 나타내는 두 가지의 원시 값 중 하나입니다. 

{
    let undefinedValue;
    console.log(undefinedValue); // 출력: undefined                                                                                                                
}
결과 확인하기
undefined

06. 심볼(Symbol)

심볼(Symbol)은 ES6(ECMAScript 2015)에서 추가된 원시 데이터 형식으로, 유일한 값을 나타내기 위한 고유한 식별자(identifier)를 생성합니다. 심볼은 다른 어떤 값과도 중복되지 않는 고유한 값을 갖습니다. 주로 객체 속성의 식별자로 사용되며, 이름 충돌을 방지하거나 의도치 않은 속성 변경을 방지하기 위해 사용됩니다. Symbol 함수를 호출하여 새로운 심볼을 생성할 수 있습니다. 

{
    let key = Symbol('key');
    let obj = {};
    obj[key] = 'value';
    console.log(obj[key]); // 출력: value                                                                                                                                  
}
결과 확인하기
value

07. 객체(Object)

{
    let person = {
        name: 'John',
        age: 30,
        city: 'New York'
    };
    console.log(person.name); // 출력: John                                                                                                                                                    
}
결과 확인하기
John

08. 배열(Array)

{
    let numbers = [1, 2, 3, 4, 5];
    console.log(numbers.length); // 출력: 5                                                                                                                                                    
}
결과 확인하기
5

09. 함수(Function)

{
    function greet(name) {
        console.log("Hello, " + name + "!");
    }
    greet("Alice"); // 출력: Hello, Alice!                                                                                                                                                  
}
결과 확인하기
Hello, Alice!
