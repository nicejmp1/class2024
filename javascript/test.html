01. 1부터 100까지 출력하기 (for문, while문, do while)

{
    for (let i = 1; i <= 100; i++) {
        console.log(i);
    }
==========================================
while 문
    {
        let num = 1;
            
        while (num <= 100) {
            console.log(num);
            num++;
        }
    }
==========================================
do while 문
    {
        let num = 1;
        do {
            console.log(num);
            num++;
        } while (num <= 100);
    }
}
결과 확인하기
for 문
1~100
while문
1~100
do while문
1~100
