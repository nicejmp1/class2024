01. 1부터 100까지 출력하기 (for문, while문, do while)

{
    for (let i = 1; i <= 100; i++) {
        console.log(i);
    }
==========================================
while 문
    {
        let num = 1;
        while (num <= 100) {
            console.log(num);
            num++;
        }
    }
==========================================
do while 문
    {
        let num = 1;
        do {
            console.log(num);
            num++;
        } while (num <= 100);
    }
}
결과 확인하기
for 문
1~100
while문
1~100
do while문
1~100

02. 0부터 100까지 짝수만 출력하기

{
    for (let i = 1; i <= 100; i++) {
        if (i % 2 == 0) {
            console.log(i);
        }
    }
        
    for (let i = 1; i <= 100; i++) {
        if (i % 2 == 1) {
            console.log(i);
        }
    }
}
결과 확인하기
짝수
홀수

03. 1부터 100까지는 짝수는 파랑색, 홀수는 빨간색 출력 

{
    for (let i = 1; i <= 100; i++) {
        if (i % 2 == 0) {
            document.write("" + i + "");
        } else {
            document.write("" + i + "");
        }
    }
}

04. 중첩 for 문을 이용하여 구구단 짜기 

{
    for (let i = 2; i <= 9; i++) {
        for (let j = 1; j <= 9; j++) {
            console.log(i + '*' + j + '=' + i * j);
        }
    }
}
결과 확인하기
구구단

05. 중첩 for 문을 이용하여 8단 구구단 짜기
